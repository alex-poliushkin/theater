# Theater Docs

This index links only to documentation and example files that exist in the
repository today.

## Learn

Start with [First Run](tutorial/01-first-run.md). It validates and runs one tiny
Theater DSL example and one YAML equivalent as the current guided entry point.

Continue through [tutorials](tutorial/index.md). After the first tutorial loop,
open [concepts](concepts/index.md) for the mental model behind the examples,
including reusable scenarios.

## Do A Task

Open [how-to pages](how-to/index.md) when you want a task recipe. Current task
routes cover validating and running flows, JSON field checks, runtime value
logging, HTTP sessions, persistent state, Theater DSL formatting and lowering,
YAML migration, reusable scenario discovery, run output, plugin registry
inspection, Go embedding, and editor tooling checks.

## Inspect An Example

Open [examples](examples/index.md) when you want complete files to inspect or
run.

## Look Up A Contract

Open [reference](reference/index.md) when you need exact contracts for
Theater DSL, YAML, CLI commands, output formats, capabilities, reports,
scenario logs, plugins, editor tooling, Go APIs, selectors, expectations, or the
source-of-truth syntax references.
