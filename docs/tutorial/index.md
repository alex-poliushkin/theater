# Tutorials

Tutorials teach one guided path.

1. [First Run](01-first-run.md) validates and runs the smallest public Theater
   example from this checkout.
2. [First Stage](02-first-stage.md) explains the file you just ran in Theater
   DSL and YAML.
3. [Edit And Fix](03-edit-and-fix.md) shows one controlled failure and the fix.
4. [Reuse A Scenario](04-reuse-a-scenario.md) moves the same small idea into a
   repo-shaped `theater/flows/` and `theater/lib/` layout.
5. [Check Values](05-check-values.md) teaches `field`, `decode(json)`, `path`,
   `log`, expectations, and a small export-backed dataflow.
6. [HTTP Flow](06-http-flow.md) calls a local HTTP fixture with a reusable
   session.
7. [Wait For Result](07-wait-for-result.md) adds persistent state and
   `eventually` polling.
